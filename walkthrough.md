## 下一步：啟用 GitHub Pages

雖然程式碼已經成功上傳，要讓網站真正能公開被瀏覽，您還需要在 GitHub 上進行以下簡單設定：

> [!IMPORTANT]
> 1. 前往您的 GitHub 儲存庫：[citriclin0422/AI_portfolio](https://github.com/citriclin0422/AI_portfolio)
> 2. 點擊上方的 **Settings** (設定) 頁籤。
> 3. 在左側選單中找到 **Pages** 並點擊。
> 4. 在 **Build and deployment** 下方的 **Source** 選擇 `Deploy from a branch`。
> 5. 在 **Branch** 的下拉選單中，選擇 `main`，資料夾保留 `/(root)`，然後點擊 **Save**。

幾分鐘後，GitHub 就會產生您的個人網址（通常為 `https://citriclin0422.github.io/AI_portfolio/`），您就可以將它分享給其他人了！

如果您需要修改網頁內容（例如修改文字、放入真實的專案圖片或您的名字），您可以直接編輯 [index.html](file:///c:/AI_class/HW2_personinfo/index.html)，完成後再跟我說，我可以幫您再次推送到 GitHub。
