## 專案介紹

本作品集包含多個實作專案，涵蓋機器學習、資料分析與網頁前端等不同領域：

- **HW1: Hello Page** - 基礎網頁練習。
- **HW2: AI 人工智慧與數據分析應用班實作專案作品集** - 展示各項實作專案的個人網站 (即本專案)。
- **HW3: txt2image by streamlit** - 使用 Streamlit 實作 AI 創意繪圖應用。
- **HW4: CRISP-DM 線性迴歸互動分析** - 運用 CRISP-DM 流程進行線性迴歸模型實作。
- **HW5: 十大機器學習演算法互動平台** - 整合多種機器學習演算法的互動平台。
- **HW6: 新創公司利潤預測與特徵選擇** - 針對新創公司數據進行特徵選擇與利潤預測。
